# Docker Nginx PHP 8 Mysql 8 Angular 13

Docker Container with
- Nginx (Latest)
- PHP 8.1
- Mysql 8.0
- Angular 13.3.7
- Redis Alpine

## Installation

Run and install development environment

```bash
$ docker-compose up
```

take it down with
```bash
$ docker-compose down
```
